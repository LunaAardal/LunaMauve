A vencord theme mainly built for women/girls without nitro



Download vencord here https://vencord.dev

how to download the theme and use it

1. download and install vencord
2. Go into settings and scroll down untill you see themes
3. click Open themes folder and drag the file you downloaded into that folder, when you're done you should see the Lunamauve theme, toggle it
